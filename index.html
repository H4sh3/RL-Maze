Reinforcement Learning a Maze

Description
The agent gets trained to reach the goal on the lower screen.
Each field stores 4 action-ratings (up,down,left,right).
The arrows indicate which action has the highest rating.
The agent explores the environment by sometimes choosing random actions.
Most of the time it performs the action with the highest rating.
After the action-ratings got adjusted, the agent reaches the goal with as little steps possible.
After each episode we run 10 episodes without rendering to speed things up.

The code can be found at Github.
